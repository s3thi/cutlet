# Documentation and Examples

**Status:** Done

- `TUTORIAL.md` — learnxinyminutes-style tutorial covering all features.
- `examples/*.cutlet` — 12 example programs, one per language feature (arithmetic, modulo-power, strings, booleans, nothing, comparison, variables, if-else, while-loop, break-continue, function-call, unary). Small, self-contained, use `say()` for output. Serve as documentation, pipeline tracer input, and lightweight feature tests.
