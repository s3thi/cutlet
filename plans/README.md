# Cutlet Plans

- **`doing/`** — work queue. Pick a task, do the work, move it to `done/` when finished.
- **`done/`** — archive of completed work. Files are numbered consecutively with 4-digit zero-padded prefixes (e.g., `0020-feature-name.md`). When moving a file from `doing/` to `done/`, use the next number after the highest existing one.
- See `AGENTS.md` for project conventions and required process.
